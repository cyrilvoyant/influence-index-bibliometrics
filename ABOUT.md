# Influence Index

The Influence Index is a scale-independent metric designed to measure institutional research concentration using the Top 2% scientists dataset.

Developed by Cyril Voyant, Jeroen Baas, Roy Boverhof, Paolo Stringari, Mathilde Marchand-Lasserre, and John P. A. Ioannidis, it complements traditional bibliometric indicators by focusing on proportional representation and logarithmic interpretability.

This repository provides MATLAB code and datasets to compute and visualize the index across institutions worldwide.
